# 3D打印作业——吉他的弟弟
## 模型文件处理
先尝试把3D scanner app上传照片得到扫描的吉他导入Blender中看看模型情况，发现有很多凹凸不平的面和残缺破面
<img width="1964" height="1142" alt="image" src="https://github.com/user-attachments/assets/9b1d949e-e8f4-48a0-b6b0-b7fcadb6c3ed" />
这样的模型很难处理，也会影响打印效果，于是我转换思路，把我用于3D scanner app中上传扫描的照片交给腾讯3D混元网站，生成一个质量比较高的吉他白模
<img width="1986" height="1202" alt="image" src="https://github.com/user-attachments/assets/3525e83f-620f-41a9-9856-316f828a54ff" />
<img width="1662" height="1202" alt="image" src="https://github.com/user-attachments/assets/c16ad030-f08f-46c3-9b5f-c2334ba962a4" />
生成的模型是完全封闭、没有非流形的错误，所以我最后采用没有重拓补的模型进行打印
<img width="1370" height="1154" alt="a869383e-c001-4d19-aca2-5702e115d3e4" src="https://github.com/user-attachments/assets/8efff00b-607e-45a0-8109-daf4e9ff70e6" />
## 模型导入Ultimaker Cura
设置好打印机后把obj模型导入到软件中，调整模型到合适的大小，用面积最大、最平整的部分接触底面，由于有悬空面，所以我添加了支撑。
